# 🐍 Projeto de Estudos em Python, Dados e Machine Learning

Bem-vindo ao repositório de exercícios e projetos em Python! Este espaço foi criado para acompanhar minha jornada de aprendizado em:

- **Python básico**
- **Estruturas de dados**
- **Lógica de programação**
- **Análise de dados**
- **Machine Learning**

## 🚀 O que este projeto faz

Aqui você encontra uma coleção organizada de exemplos e exercícios práticos que cobrem desde conceitos iniciais até aplicações mais avançadas em ciência de dados.

A estrutura do repositório reflete um fluxo de aprendizado gradual:

1. **`1-basico/`**: fundamentos de Python, lógica, loops, funções e operações matemáticas.
2. **`2-intermediario/`**: classes, leitura e escrita de arquivos, uso de módulos e manipulação de dados.
3. **`3-dados & libs/`**: análise de dados com `pandas`, visualização, regressão e estatística aplicada.
4. **`4-machine learning/`**: primeiras experimentações com modelos de aprendizado supervisionado.

## 🧠 Por que a sequência de conhecimento importa

A sequência está pensada para fortalecer a base antes de avançar:

- Primeiro construo a lógica e os conceitos de Python.
- Depois aplico essa base em projetos e manipulação de dados.
- Finalmente exploro machine learning com modelos e análises mais robustas.

Isso garante que cada etapa seja compreendida com clareza antes de introduzir novas ferramentas ou conceitos.

## 🔄 Evolução contínua

Este repositório está em constante atualização. Novos exemplos, modelos e estudos são adicionados com frequência para acomodar:

- novas técnicas de machine learning
- experimentos com diferentes bibliotecas
- exercícios práticos de análise de dados
- apoio ao estudo contínuo e à melhoria dos conhecimentos
